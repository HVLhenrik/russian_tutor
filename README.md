# Russian Declension Tutor

## Overview
The Russian Declension Tutor is a terminal-based application designed to help users learn how to decline Russian nouns and adjectives. The program provides a comprehensive list of Russian words along with their declensions across different grammatical cases, facilitating an interactive learning experience.

## Features
- Declension of Russian nouns and adjectives
- Interactive quizzes to test knowledge
- User-friendly terminal interface
- Comprehensive database of nouns and adjectives

## Installation
To set up the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd russian-declension-tutor
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage
To run the application, execute the following command in the terminal:
```
python src/main.py
```

Follow the on-screen instructions to learn and practice the declension of Russian nouns and adjectives.
